# BimBam_Buy_desafio
Este proyecto implementa un flujo de trabajo automatizado para clasificar consultas de clientes, responder preguntas frecuentes mediante RAG (FAISS + Gemini Embeddings) y escalar solicitudes complejas a tickets de soporte técnico.

## 🌐 Acceso en línea
Puedes probar la aplicación directamente desde Streamlit Cloud sin necesidad de instalación:
👉 [Abrir aplicación](https://bimbambuydesafio-cweagv4gp3nnxpjffef3st.streamlit.app/)

## 🚀 Instalación local (opcional)
Si deseas ejecutar el proyecto en tu entorno local:
```bash
git clone https://github.com/alanbubu18-design/bimbambuy-agent.git
cd bimbambuy-agent
pip install -r requirements.txt
streamlit run app.py
